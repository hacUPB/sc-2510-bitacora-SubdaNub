### Transforma el programa en alto nivel anterior para que utilice un ciclo for en vez de un ciclo while.
Ciclo While:
```c++
    int i = 1;
    int sum = 0;
    While (i <= 100){
        sum += i;
        i++;
    }
```
Ciclo For:
```c++
    int sum = 0;
    For (int i,i <= 100, i++){
        sum += i;
    }
```
